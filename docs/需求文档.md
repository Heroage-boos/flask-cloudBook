## 书籍公共 API 推荐

### 免费书籍搜索 API

| API | 说明 | 特点 |
|-----|------|------|
| **Open Library API** | 开放图书馆 | 完全免费，数据丰富，支持 ISBN 查询 |
| **Google Books API** | 谷歌图书 | 全球最大书库，功能完善 |
| **Douban API (非官方)** | 豆瓣图书 | 中文书籍数据，支持登录用户数据 |
| **Amazon Books API** | 亚马逊 | 商业书籍数据丰富 |

### 推荐使用方案

#### 方案一：Open Library + 豆瓣（推荐）
```
推荐理由：
- Open Library 完全免费、开源，无调用限制
- 豆瓣提供优质中文书籍数据
- 两者结合可覆盖中英文书籍
```

#### 方案二：仅使用 Open Library
```
适用场景：面向国际用户或英文书籍为主
优点：完全免费、数据质量高、无需申请密钥
缺点：中文书籍数据相对较少
```

### API 详情

#### 1. Open Library API
```python
# 搜索书籍
GET https://openlibrary.org/search.json?q={关键词}

# 按 ISBN 查询
GET https://openlibrary.org/isbn/{isbn}.json

# 获取书籍详情
GET https://openlibrary.org/works/{work_id}.json

# 获取书籍封面
GET https://covers.openlibrary.org/b/id/{cover_id}-M.jpg
```

#### 2. Google Books API
```python
# 需要 API Key (免费额度: 1000次/天)

# 搜索书籍
GET https://www.googleapis.com/books/v1/volumes?q={关键词}&key={API_KEY}

# 按 ISBN 查询
GET https://www.googleapis.com/books/v1/volumes?isbn={isbn}&key={API_KEY}
```

#### 3. 豆瓣图书 (需要登录/Token)
```
接口有限制，建议配合 Open Library 使用作为补充
主要优势：中文书籍数据完整
```

## 注意事项

1. **版权合规**：确保电子书版权合法，避免侵权
2. **文件安全**：电子书文件加密存储，防止盗链
3. **支付安全**：使用正规支付渠道，防范欺诈
4. **性能考虑**：大文件下载建议使用 CDN 或云存储
5. **用户体验**：阅读器兼容多格式，支持多设备同步
